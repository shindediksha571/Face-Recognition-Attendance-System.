# Face Recognition Attendance System

## Project Description
This project is a Face Recognition Attendance System developed using Python, OpenCV, and the face_recognition library. It detects registered faces through a webcam and automatically marks attendance in a CSV file.

## Features
- Face Recognition using OpenCV
- Automatic Attendance Marking
- Unknown Face Detection
- Attendance stored in CSV format
- Real-time Webcam Detection

## Technologies Used
- Python
- OpenCV
- face_recognition
- NumPy
- Pandas

## Installation

```bash
pip install -r requirements.txt
```

## Run the Project

```bash
python app.py
```
